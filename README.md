# 记事本APP

一个简单的记事本应用程序，支持创建、编辑、保存和查看笔记。

## 功能特点

- ✅ 创建新笔记
- ✅ 编辑现有笔记
- ✅ 保存笔记到本地存储
- ✅ 查看所有笔记列表
- ✅ 删除笔记
- ✅ 搜索笔记内容
- ✅ 笔记分类

## 技术栈

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## 快速开始

### 在线访问
应用已部署到 GitHub Pages，可以直接访问：
- https://fengqingxiao.github.io/notepad-app/

### 运行方式
1. 克隆项目
2. 使用浏览器打开 `index.html` 即可使用

### 本地运行
```bash
git clone https://github.com/fengqingxiao/notepad-app
cd notepad-app
# 直接使用浏览器打开 index.html
```

## 使用说明

1. 点击"新建笔记"按钮创建新笔记
2. 输入标题和内容
3. 点击"保存"按钮保存笔记
4. 点击笔记列表中的笔记进行编辑
5. 点击"删除"按钮删除笔记
6. 使用搜索框查找笔记内容
