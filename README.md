# 记事本APP

一个简单而强大的记事本应用程序，使用纯HTML/CSS/JavaScript构建，无需后端服务器，完全离线使用。

## 🌟 功能特点

- ✅ **创建新笔记** - 快速创建新的记事
- ✅ **编辑现有笔记** - 随时修改已保存的笔记
- ✅ **本地存储** - 使用LocalStorage保存数据，完全离线
- ✅ **笔记分类** - 支持工作、学习、生活、备忘、灵感等类别
- ✅ **搜索功能** - 快速查找笔记内容
- ✅ **统计信息** - 显示笔记总数和各类别统计
- ✅ **键盘快捷键** - Ctrl+N新建笔记，Ctrl+S保存笔记
- ✅ **导出功能** - 导出笔记为JSON文件
- ✅ **响应式设计** - 适配各种屏幕尺寸
- ✅ **美观界面** - 简洁现代的UI设计

## 📱 使用说明

### 基本操作
1. **新建笔记**: 点击"新建笔记"按钮或按Ctrl+N
2. **编辑笔记**: 在左侧笔记列表中选择笔记进行编辑
3. **保存笔记**: 点击"保存笔记"按钮或按Ctrl+S
4. **删除笔记**: 选择笔记后点击"删除笔记"按钮
5. **搜索笔记**: 使用搜索框查找特定笔记
6. **分类筛选**: 点击类别标签筛选笔记

### 快捷键
- **Ctrl+N**: 新建笔记
- **Ctrl+S**: 保存笔记
